# anarchy-benchmarks

## Setup (MacOS):

```
# create local python environment
python3 -m venv env

# activate local python environment
source env/bin/activate

# install pytorch
pip3 install torch torchvision torchaudio

# install LLM-VM
git clone https://github.com/anarchy-ai/LLM-VM.git
cd LLM-VM
pip3 install .
cd ..
```
